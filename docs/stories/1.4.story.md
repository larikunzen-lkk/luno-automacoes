# Story 1.4 — LGPD Cookie Banner

**Epic:** 1 — Fundação & Homepage
**Status:** InProgress
**Assignee:** @dev (Dex)

---

## Goal

Implementar banner de consentimento LGPD que controla o disparo do Meta Pixel.
O pixel só pode ser inicializado após consentimento explícito. Preferência
salva em localStorage. Incluir página /privacidade com política básica.

## Acceptance Criteria

- [ ] AC1: Banner aparece na primeira visita (sem consentimento prévio)
- [ ] AC2: Botões "Aceitar" (primário) e "Recusar" (ghost) — nunca `disabled`
- [ ] AC3: Preferência salva em localStorage (`luno_cookie_consent`)
- [ ] AC4: Meta Pixel só inicializa após aceitar — NUNCA antes
- [ ] AC5: Banner não bloqueia scroll nem interação com o conteúdo
- [ ] AC6: Banner reaparece se localStorage for limpo
- [ ] AC7: Página /privacidade com política básica e link de volta
- [ ] AC8: Link "Saiba mais" no banner aponta para /privacidade
- [ ] AC9: WCAG AA — focus trap não aplicado (banner não modal)
- [ ] AC10: Build + lint + typecheck passando
