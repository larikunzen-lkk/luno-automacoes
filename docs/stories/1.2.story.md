## Goal

Implementar a homepage completa da Luno Automações com todas as seções definidas na front-end-spec: Header sticky, Hero, Calculadora de ROI interativa, Como Funciona, Cases (placeholder), Preview de Pacotes e CTA Final + Footer.

## Acceptance Criteria

- [ ] AC1: Header sticky com logo, link "Serviços" e botão CTA. Mobile: logo + CTA apenas
- [ ] AC2: Hero com headline/subheadline da Luno, CTA principal e logos de integração (Make, n8n, Zapier, Google)
- [ ] AC3: Calculadora de ROI — slider (mobile) e input (desktop), resultado em tempo real, aria-live="polite"
- [ ] AC4: Seção "Como Funciona" com 3 cards/passos
- [ ] AC5: Cases placeholder com 2 cards (dados simulados com disclaimer)
- [ ] AC6: Preview de pacotes com link para /servicos
- [ ] AC7: CTA Final destacado com botão para /diagnostico
- [ ] AC8: Footer com logo, tagline, links e redes sociais
- [ ] AC9: Sticky CTA mobile (barra inferior após 300px scroll)
- [ ] AC10: WCAG AA — todos elementos interativos com focus ring
- [ ] AC11: Responsivo — mobile 375px e desktop 1280px
