## Goal

Implementar skip link de acessibilidade, melhorar gestão de foco no header, garantir navegação 100% funcional por teclado e corrigir quaisquer gaps de WCAG AA identificados após deploy da homepage.

## Acceptance Criteria

- [ ] AC1: Skip link "Pular para o conteúdo" visível ao foco, antes do header
- [ ] AC2: `<main id="main-content">` como âncora do skip link
- [ ] AC3: Navegação completa por Tab/Shift+Tab sem armadilhas de foco
- [ ] AC4: Header mobile — botão hamburger com aria-expanded e aria-controls
- [ ] AC5: Menu mobile abre/fecha com animação suave, fecha ao pressionar Escape
- [ ] AC6: Active link state visível no header (aria-current="page")
- [ ] AC7: Lighthouse Accessibility mantido em 100
